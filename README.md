# Cafeteria-Management-System

A comprehensive C-based cafeteria management system designed for Bangladesh University of Professionals (BUP) that handles multiple cafeterias, student accounts, orders, inventory management, and payment processing.

## Features

1. Login & Registration: Students and admins can securely create accounts and log in to access their respective dashboards.
2. Balance Management: Students can add money to their account balance for making purchases.
3. Food Ordering System: Students can browse menus from multiple cafeterias and place orders easily.
4. Order & Admin Management: Admins can manage orders, students, menus, prices, and inventory from a centralized dashboard.
5. Order Tracking & Payments: Orders include token numbers, status tracking (Pending/Paid/Cancelled), and cash payment handling with receipt generation.

## Programming Language
- C
